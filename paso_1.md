
<a href="./requisitos.md">atrás</a> - Paso 1: Crear el Medio de Instalación y Arrancar - <a href="./paso_2.md">siguiente</a>

1.  **Crea el USB de Arranque:** Utiliza Rufus o Ventoy para escribir la imagen ISO de Ubuntu Server 24.04 en tu unidad USB.
2.  **Arranca desde el USB:** Conecta la unidad USB al servidor. Reinicia el servidor e ingresa al menú de arranque (generalmente presionando `F2`, `F10`, `F12` o `Supr`) para seleccionar la unidad USB como dispositivo de arranque.
3.  **Selecciona el Instalador:** En el menú de inicio, selecciona la opción **"Install Ubuntu Server"**.
