Asegúrate de tener lo siguiente antes de empezar:

1.  Una **máquina** con el hardware compatible (como mínimo 2 GB de RAM, 25 GB de disco).
2.  La **imagen ISO** de Ubuntu Server 24.04 LTS, <a target="_blank" href="https://ubuntu.com/download/desktop/thank-you?version=24.04.3&architecture=amd64&lts=true">Descarga</a>
3.  Una **unidad USB** de al menos 8 GB.
4.  Un software para crear una unidad USB de arranque (como **Ventoy** o **Rufus**).
