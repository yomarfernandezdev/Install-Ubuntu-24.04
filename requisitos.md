Asegúrate de tener lo siguiente antes de empezar:

1.  Una **máquina** con el hardware compatible (como mínimo 2 GB de RAM, 25 GB de disco).
2.  La **imagen ISO** de Ubuntu Server 24.04 LTS, <a target="_blank" href="https://releases.ubuntu.com/24.04.3/ubuntu-24.04.3-desktop-amd64.iso?_gl=1*7pb76y*_gcl_au*MTIwMjk2MDc2Mi4xNzYxNzQzNzkw">Descarga</a>
3.  Una **unidad USB** de al menos 8 GB.
4.  Un software para crear una unidad USB de arranque (como **Ventoy** o **Rufus**). <a href="https://rufus.ie/es/">Rufus</a>
