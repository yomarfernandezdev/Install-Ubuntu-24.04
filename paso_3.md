Esta es una parte crucial.

1.  **Modo de Almacenamiento:** Elige una de las siguientes opciones:
    * **"Use an entire disk" (Usar todo el disco):** La forma más sencilla. El instalador particionará automáticamente todo el disco para ti. **(Recomendado para principiantes)**.
    * **"Custom storage layout" (Diseño de almacenamiento personalizado):** Si necesitas configurar RAID, LVM o particiones específicas.
2.  **Selección de Disco:** Selecciona el disco duro o SSD donde deseas instalar el sistema operativo. **¡Asegúrate de elegir el disco correcto, ya que todos los datos en él serán borrados!**
3.  **Resumen de Almacenamiento:** Revisa el resumen de las particiones propuestas. Si estás de acuerdo, selecciona **"Done"**. Confirma la advertencia de que el disco será formateado.
